The purpose of this Code of Conduct is to identify standards of behaviour by which all volunteers registered with EMBC must abide. Volunteers who violate the Code of Conduct may have their registration revoked or suspended.

1.02 PUBLIC SAFETY LIFELINE VOLUNTEER CODE OF CONDUCT
1.02 PUBLIC SAFETY ANNEX

If a volunteer is unclear on any element of the code of conduct, the volunteer will seek out clarity from their PSLV Organization, Local Government or an EMBC regional office.
